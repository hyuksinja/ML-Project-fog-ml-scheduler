Novel Contributions (Patentability)
1
HCFE
Heterogeneous-Context Feature Engineering
18 features across task, node, and 7 novel interaction dimensions (cpu_demand_ratio, mem_pressure, net_intensity, effective_mips, deadline_slack, load_mem_interact, tier_task_match). No prior fog-cloud paper combines all three dimensions.
2
UQE
Uncertainty-Quantified Ensemble
Bootstrap resampling (n=30) over RF+GB+SVR produces per-prediction confidence intervals. Outputs point estimate AND standard deviation σ. No prior execution-time predictor for fog-cloud provides calibrated uncertainty estimates.
3
UASP
Uncertainty-Aware Scheduling Policy
Selects node minimising: score = pred_time + α·std. α is tunable: α=0 recovers ML-Greedy, α>0 is conservative, α<0 is exploratory. This single formula generalises all existing ML-based scheduling policies.
4
CSD
Context-Shift Detector
Sliding-window KL-divergence monitor compares incoming task stream against training distribution. Automatically flags retrain_needed when max feature KL exceeds threshold. Creates a self-healing closed-loop scheduling system.
